# 文档中心

单一文档根。开发只认 **features** 与 **requirements**；**specs** 为设计规格；**archive / reference** 仅溯源与调研。

```text
docs/
├── features/charts/     ← 图表定稿（唯一依据）
├── requirements/        ← 功能需求
├── specs/               ← 设计规格
├── archive/             ← 合并前 features 溯源
└── reference/           ← LabKey / Benchling 调研原文
```

| 目录 | 说明 |
| --- | --- |
| [features/charts/](./features/charts/) | LabKey + Benchling 合并后的图表功能点 |
| [requirements/](./requirements/) | 表（vxe-table）+ 图一体化等 |
| [specs/](./specs/) | Insight 框架设计、合并过程设计 |
| [archive/](./archive/) | 旧 `charts` / `benchling-charts` 原文 |
| [reference/](./reference/) | 官方文档摘录（非需求） |
