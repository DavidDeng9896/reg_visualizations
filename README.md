# Insight Analysis / reg_visualizations

纯前端科学数据工作空间：导入 CSV、合并表、转换视图、流程图导航、ECharts 科学图表。

## 快速开始

```bash
npm install
npm run dev      # http://localhost:5173
npm test
npm run build
```
