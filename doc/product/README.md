# 产品文档

本目录为**产品开发依据**，后续设计与实现以此为准。

```text
product/
├── features/          ← 功能点细则（已确认范围）
│   └── charts/        ← 五类科学图表
└── requirements/      ← 功能需求（场景/一体化/分期）
```

| 目录 | 作用 |
| --- | --- |
| [features/charts/](./features/charts/) | 图表 Type / Layout / 导出等已确认功能点 |
| [requirements/](./requirements/) | 数据表 + 图表等需求文档 |

参考调研原文（非开发依据）：[`../reference/`](../reference/)
